# Product Overview

Super Kiro World is a 2D platformer game built for browser environments. The game features a flying Kiro character that navigates through platform-based levels, collecting gems and power-ups while avoiding obstacles.

## Core Features
- Flying character mechanics with directional movement
- Platform-based level design with collision detection
- Collectible system (gems, power-ups)
- Score tracking and lives system
- Smooth camera following player movement
- Level completion objectives (reach the flag)
- Game over and restart functionality

## Target Audience
- Workshop participants learning game development
- Developers exploring HTML5 Canvas and vanilla JavaScript
- Players enjoying retro-style platformer games

## Brand Integration
- Uses Kiro brand colors (Orange-500 #FF8C00 primary, Orange variants)
- Incorporates Kiro logo as game sprite
- Maintains dark theme aesthetic consistent with Kiro branding
