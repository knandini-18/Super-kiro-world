# User Context - Super Kiro World Game

## User Preferences
- **Language**: Vanilla JavaScript with HTML5 Canvas
- **Game Type**: 2D Platformer (Super Kiro World)
- **Style**: Retro pixel art aesthetic with Kiro branding
- **Physics**: Balanced jumping mechanics (gravity: 0.5, jumpPower: 12, speed: 5)

## Game Requirements
- Platform-based level design
- Kiro character with left/right movement and jumping
- Scrolling camera following player
- Collectible items throughout level
- Collision detection for platforms and collectibles
- Score counter and lives system
- Level completion and restart functionality
- Friction for realistic movement feel
- Responsive platforming controls

## Technical Specs
- HTML5 Canvas rendering
- 60 FPS target performance
- Smooth camera scrolling
- Proper ground detection
- Browser-compatible vanilla JS

## Visual Style
- Use Kiro brand colors (Orange-500 #FF8C00 for accents)
- Dark theme with Black-900 background
- Kiro logo as game sprite
- High contrast for accessibility
